### SnifLeaf – Native Network Inspector
*Apr 2025 – Present | macOS & iOS*
**[View on GitHub](https://github.com/hgq287/SnifLeaf)**

SnifLeaf is an open-source HTTPS proxy for developers and QA who need to inspect traffic without running Charles. Built with **SwiftUI** and **NetworkExtension**.

* Network debuggers tend to be heavy; this one stays native and small.
* **Tech:** Swift, SwiftUI, Combine, NetworkExtension.

### dttool – Developer Utility Toolkit
*Mar 2025 – Present | Python & API*
**[View on GitHub](https://github.com/hgq287/dttool)**

Python toolkit and REST API for the small format conversions that show up every week in data work.

* Converts JSON, CSV, Excel, Feather; tracks API history; RGB/HEX and note helpers on the side.
* **Tech:** Python, FastAPI, Pandas.
